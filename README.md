## ğŸ“˜ Introduction  
ClaimEase CMCS is an **ASP.NET Core** web application designed to streamline the process of managing claims for independent contractors (IC), such as lecturers.  
This enhanced version introduces functional capabilities for claim submission, document management, and status tracking, allowing admins to manage and update claims efficiently.

## ğŸ—ï¸ Architecture  
The ClaimEase CMCS application follows the **Model-View-Controller (MVC)** design pattern:  
- **Models:** Define the data structure for claims, users, and related entities.  
- **Views:** Provide the user interface components that interact with the data and display information.  
- **Controllers:** Handle the flow of data between the model and the view, processing user input and updating the model accordingly.

## â“ FAQs  
**Q1:** How do I submit a claim?  
A: Navigate to the "Submit Claim" page, enter your claim details, and upload any supporting documents. Click "Submit" to send your claim for processing.  

**Q2:** How can I track the status of my claim?  
A: Go to the "View Claims" section to see the status of all your submitted claims.  

**Q3:** What should I do if my claim is rejected?  
A: Review the feedback provided and make any necessary adjustments before resubmitting your claim.  

**Q4:** How do I reset my password?  
A: Contact your admin or use the "Forgot Password" feature on the login page.  

**Q5:** Can I update my profile information?  
A: Yes, you can update your profile information in the "Manage Profile" section.
